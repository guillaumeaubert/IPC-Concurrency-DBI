IPC-Concurrency-DBI

IPC::Concurrency::DBI controls how many instances of a given application are
allowed to run in parallel, using DBI as the IPC method.

INSTALLATION

To install this module, run the following commands:

	perl Build.PL
	./Build
	./Build test
	./Build install
